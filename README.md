# Java Console Chat App
🌐A simple multi-threaded localhost chat app created with JAVA. The following methods are implemented:
1. User login / registration for first time users.
2. Broadcast messages to all
3. Private messages to individual users.

## 📖Formats to use:
1. /msg <username:Text> <messagebody:Text> : Sends private message to the given username ( if online)
2. /users : Fetch all the users ( online , offline )
3. <message:Text> : Broadcast to all.
